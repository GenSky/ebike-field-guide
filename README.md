# E-Bike Field Guide

A static GitHub Pages site about choosing, riding, maintaining, and comparing electric bikes.

## Local Preview

Open `index.html` in a browser, or run a small static server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

This repo is designed to deploy with GitHub Pages from the `main` branch root.

After the repository is pushed to GitHub:

1. Open the repository settings.
2. Go to **Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Set **Branch** to `main` and folder to `/root`.
5. Save the settings.
